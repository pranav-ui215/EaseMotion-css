# Ease Command Palette

## Overview

Ease Command Palette is a reusable command menu inspired by modern code editors such as VS Code. It features a search bar, animated command list, and keyboard shortcut badges using pure HTML and CSS.

---

## Features

- ⌨️ Command palette UI
- 🔍 Search field
- ✨ Animated popup
- 🎯 Hover highlight
- ⌨️ Keyboard shortcut badges
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---
